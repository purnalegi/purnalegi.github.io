Membuat File CSS Untuk Situs Di GitHub

Dari dasbor repository klik Add file.
Klik Create new file.
Sematkan nama file, contoh: css/purnalegizon-style.css.
Klik Commit new file.

Mengunggah Favicon Untuk Situs Di GitHub

Dari dasbor repository klik Add file.
Klik Create new file.
Sematkan nama file, contoh: gambar/readme.md.
Klik Commit new file.
Klik Add file.
Klik Upload files.
Klik choose your files.
Klik Commit changes.

Membuat Subdomain Gratis Di GitHub

Dari dasbor akun klik Create repository.
Sematkan nama repository, contoh: purnalegi.github.io.
Sematkan deskripsi repository, contoh: Repositori purnalegi.github.io.
Klik kotak Add a README file.
Klik Create repository.
Klik Add file.
Klik Create new file.
Sematkan nama file, contoh: index.html.
Klik Commit new file.

Membuat Akun Gratis Di GitHub

Ke https://github.com/.
Klik Sign up.
Sematkan email kemudian klik Continue.
Sematkan password kemudian klik Continue.
Sematkan username kemudian klik Continue.
Sematkan y kemudian klik Continue.
Klik Start puzzle kemudian selesaikan tantangannya.
Klik Create account.
Sematkan launch code yang diterima via email.
Pilih Just me dan Student kemudian klik Continue.
Centang kotak Community kemudian klik Continue.
Klik Continue for free.
