Membuat Folder Repositori Di GitHub

Membuat folder di dalam repositori bisa dengan cara berikut:

Klik Add file.

Klik Create new file.

Beri nama file, contoh: gambar/favicon.gif.

Klik Commit new file.

Klik folder yang baru dibuat.

Klik Add file.

Klik Upload files.

Klik choose your files.

Klik Commit changes kalo udah selesai.
