Menghosting Situs Di GitHub

Untuk bisa menghosting situs di GitHub harus punya akun lebih dulu.

Kalo belom punya, silakan lihat yang ini: Membuat Akun (Versi Gratis) Di GitHub.

Kalo udah punya, silakan lanjut.

Dari dasbor klik Create Repository.

Beri nama repositori tersebut, contoh: purnalegi.github.io.

Beri deskripsi repositori tersebut (opsional).

Klik kotak Add a README file.

Klik Create repository.

Klik Add file.

Klik Create new file.

Beri nama file, contoh: index.html.

Isi index.html dengan contoh dibawah ini.

<!DOCTYPE html>
<html lang="id-ID">
    <head>
        <meta charset="utf-8"/>
        <meta name="viewport" content="width=device-width"/>
        <title>Purnalegizon</title>
        <meta name="description" content="Beranda purnalegi.github.io"/>
    </head>
    <body>
        <h1>Purnalegizon</h1>
        <h2>Halo, Dunia!</h2>
        <p>Selamat Datang<br/>
        Dan<br/>
        Terima Kasih Atas Kunjungan Kamu</p>
        <p>2022 <a href="https://purnalegi.github.io/">Purnalegizon</a> Oleh <a href="https://github.com/purnalegi/" rel="me">purnalegi</a><br/>
        Situs Ini Dihosting Di <a href="https://github.com/">GitHub</a></p>
    </body>
</html>

Klik Commit new file.

Mengakses situs via browser. Contoh URL: https://purnalegi.github.io/.
